# MVP Scoping Assistant

Help founders define and scope their minimum viable product.

## Description

This skill guides you through the process of scoping an MVP:

1. **Problem Definition** - Clearly articulate the problem you're solving
2. **Target User** - Define your ideal first customer
3. **Core Features** - Identify the 3-5 features essential for launch
4. **Success Metrics** - Define how you'll measure MVP success

## Usage

Ask me to help you scope your MVP idea. I'll ask clarifying questions and help you narrow down to the essential features.

## Example Prompts

- "Help me scope an MVP for a fitness tracking app"
- "What features should I include in my MVP for a B2B invoicing tool?"
- "Is my feature list too big for an MVP?"
